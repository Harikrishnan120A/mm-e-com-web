# Minimalist Merchants

**Design-led 3D printing & fabrication studio** based in Rajapalayam, Tamil Nadu, India.

> Clarity over complexity. Intent over decoration. Process over shortcuts.

## About

Minimalist Merchants (MiniMerchStudio) is a sole proprietorship founded on December 25, 2025. We design and fabricate objects with clarity and restraint — making things that need to exist, and making them well.

We don't chase trends or volume. We chase precision.

## What We Make

- **Modular Objects** — Desk organizers, perpetual calendars, workspace systems
- **Custom Builds** — NFC cards, QR keychains, custom enclosures, branded accessories
- **Small-Batch Fabrication** — Functional prototypes and small production runs

## Materials & Process

## Tech Stack

Single-page website built with:

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, fluid typography (`clamp()`), CSS Grid, Flexbox, keyframe animations
- **Vanilla JavaScript** — No frameworks or dependencies
- **Google Fonts** — DM Serif Display + DM Sans

### Features

- Branded preloader with font loading detection
- Film grain SVG noise overlay
- Dual-layer custom cursor (desktop)
- Auto-hide navbar with backdrop blur
- Word-by-word hero heading reveal
- Infinite scrolling marquee band
- Product grid with hover micro-interactions
- Product detail modal
- Cart drawer with localStorage persistence
- Toast notification system
- Animated stat counters (IntersectionObserver)
- Newsletter form with success state
- Mobile-responsive with hamburger menu
- Scroll progress bar & back-to-top button
- Keyboard accessible (ESC to close, focus-visible states)
